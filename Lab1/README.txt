Description
This is a linker. It is set to have a memory of 200 words. It resolves the target relative addresses and external addresses. There is error detection in this linker according to the course requirement. And the maximum length of a symbol is set to 8.

Usage
Input through keyboard and the result is shown on the screen.

Notice
The input format should be strictly following the standard input format, which means numbers should be numerical without mixing with alphabets.

Compile
compile through code "g++ linker.cpp -o linker"
run by "./linker"
